# Island Item

Represents a single item used in the Island Planner game mode.

|      Property      |                             Type                              |                                     Description                                     |
| :----------------: | :-----------------------------------------------------------: | :---------------------------------------------------------------------------------: |
|       **id**       |                           `number`                            |                            Unique ID of the island item.                            |
|      **name**      |                           `string`                            |                          Display name of the island item.                           |
|      **icon**      |                           `string`                            |                         Icon this item uses (lower cased).                          |
|  **description**   |                           `string`                            |                  Description of what this item does or represents.                  |
|      **type**      |             [`IslandItemType`](#island-item-type)             |                       The category/type of this island item.                        |
|     **rarity**     |                [`Rarity`](../common.md#rarity)                |                             Rarity of the island item.                              |
|     **limit**      |                           `number`                            |                           Limit per slot, 0 is infinite.                            |
|   **max_slots**    |                           `number`                            |                         Maximum slot count, 0 is infinite.                          |
|   **influence**    |                           `number`                            |                            Item demand, used for sales.                             |
|  **order_price**   |                           `number`                            |                             Item price, used for sales.                             |
|   **categories**   |                          `string[]`                           |      Tags / categories this item belongs to (used for filtering and grouping).      |
|  **can_convert**   |                           `boolean`                           |      Whether this item can be converted into season points or other resources.      |
| **convert_points** |                           `number`                            |                 Amount of points gained when converting this item.                  |
| **craftable_at?**  |                          `number[]`                           |          **(Optional)** List of place IDs where this item can be crafted.           |
|  **sellable_at?**  |                          `number[]`                           |            **(Optional)** List of place IDs where this item can be sold.            |
|    **used_at?**    |                          `number[]`                           |         **(Optional)** List of place IDs where this item is used/consumed.          |
| **sub_attribute**  | [`{ attr: IslandAttr; value: number } \| null`](#island-attr) | **(Optional)** Secondary Island attribute used for sales, value is the coefficient. |

# Island Item Type

Island Item Type is a numeric value numeric value with the ranges `[1, 5]` and `[9000, 9001)` where each
number represents a category of Island Planner items:

| Value |    Label     |               Description                |
| :---: | :----------: | :--------------------------------------: |
|   1   |   Material   | Generic material used for crafting/etc.  |
|   2   |     Prop     | General-purpose prop used in Island mode |
|   3   | Special Prop |     Special prop with unique effects     |
|   4   |     EXP      |    Item that grants Island experience    |
|   5   |   EXP Book   |  Book item that grants more experience   |
| 9000  | Season Point | Item representing season point currency  |

# Island Attr

Island attributes represent the various skills used in Island Planner:

- `"plant"` – Farming related tasks.
- `"collect"` – Gathering resources.
- `"conserve"` – Husbandry tasks.
- `"cooking"` – Cooking tasks.
- `"manage"` – Management/sales tasks.
- `"machinery"` – Machinery/Production tasks.

| Value |  Label  |                       Description                       |

| **description** |                     `string`                      |                   Description of the task objective.                    |
| **conditions**  | [`IslandTaskCondition[]`](#island-task-condition) | Prerequisite conditions that must be satisfied before the task appears. |
|    **event**    |                     `boolean`                     |           Whether this task is tied to a limited-time event.            |
|   **targets**   |    [`IslandTaskTarget[]`](#island-task-target)    |               Detailed target requirements (e.g. counts).               |
|   **rewards**   |      [`ItemDrop[]`](../common.md#item-drop)       |                    Rewards granted upon completion.                     |
|    **type**     |       [`IslandTaskType`](#island-task-type)       |                Task category (main, daily, event, etc.).                |

# Island Task Condition Type

Condition Type is a numeric value with ranges `[1, 9]` and `[99, 100)` where each
value describes the type of prerequisite:

| Value |          Label          |                         Description                         |
| :---: | :---------------------: | :---------------------------------------------------------: |
|   1   |       Achievement       |               Requires certain achievements.                |
|   2   |          Task           |             Requires completion of other tasks.             |
|   3   |         Ability         | Requires certain Island abilities/tech (server side check). |
|   4   |     Mutex (unused)      |          Reserved for mutual-exclusion conditions.          |
|   6   |      Own Any Item       |           Requires owning any of a set of items.            |
|   7   |      Own All Items      |           Requires owning all of a set of items.            |
|   8   | Own Any Commander Dress |      Requires owning any of listed commander dresses.       |
|   9   | Own All Commander Dress |      Requires owning all of listed commander dresses.       |
|  99   |       Impossible        |                Condition that cannot be met.                |

# Island Task Type

Task Type is a numeric value range `[1, 9]` where each
value describes what kind of task this is:

| Value |     Label      |        Description         |
| :---: | :------------: | :------------------------: |
|   1   |      Main      |      Main story task.      |
|   2   |     Branch     |     Side/Branch task.      |
|   3   |     Daily      |   Daily repeatable task.   |
|   4   |     Weekly     |  Weekly repeatable task.   |
|   5   | Event (Branch) | Event-limited branch task. |
|   6   | Event (Daily)  | Event-limited daily task.  |
|   7   | Event (Weekly) | Event-limited weekly task. |
|   8   |     Season     |    Season-related task.    |
|   9   |     Hidden     |    Hidden/secret task.     |
